# Gentlelion

> 젠틀몬스터 공식 홈페이지 클론 코딩 프로젝트

<br />

## 프로젝트 소개

바닐라 JavaScript를 중심으로 젠틀몬스터 홈페이지를 클론 코딩한 팀 프로젝트입니다.
REST API 기반 데이터 처리, CRUD 구현, 시맨틱 마크업 설계를 목표로 하며,
유저 페이지와 관리자 페이지를 분리 배포하는 모노레포 구조로 구성되어 있습니다.

- **기간:** 2026년 3월 26일 ~ 4월 20일
- **유저 페이지:** [메인](https://gentlelion.vercel.app/)
- **어드민 페이지:** [관리자](https://gentlelion-adminmin.vercel.app/)

## 프로젝트 구조

```

├── packages/
│   ├── user/        # 유저 페이지
│   ├── admin/       # 관리자 페이지
```

## Git 컨벤션

### 브랜치 전략

| 브랜치 | 설명 |
|--------|------|
| `main` | 배포 브랜치 |
| `develop` | 통합 브랜치 |
| `feature/이슈번호-설명` | 기능 개발 브랜치 |

### 커밋 메시지

| 타입 | 설명 |
|------|------|
| `feat` | 새로운 기능 추가 |
| `fix` | 버그 수정 |
| `docs` | 문서 수정 |
| `design` | UI/스타일 수정 |
| `refactor` | 코드 리팩토링 |
| `chore` | 빌드, 설정 변경 |
